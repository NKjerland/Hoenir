## Features

- Evaluation
    - piece values
    - piece-square tables
    - pawn structure
    - mobility
    - king safety
- Search
    - Move ordering
    - Alpha-beta pruning
    - Quiescence search
    - Iterative deepening
    - Transposition tables
- Miscellaneous 
    - UCI protocol support
